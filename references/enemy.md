# 적/몬스터 디자이너 (Enemy Designer)

전투 장면에 등장할 적, 몬스터, 보스의 디자인 이미지를 미드저니로 생성한다.

---

## 적 생성 플로우

### 1단계: 정보 수집
사용자에게 AskUserQuestion으로 확인한다:
- **크기**: 소형(드론/벌레) / 인간형 / 대형(3~5m) / 초대형(건물급)
- **유형**: 기계 / 생물 / 언데드 / 마법 / 하이브리드
- **위협 등급**: 잡졸 / 중형 / 엘리트 / 보스
- **특수 능력**: 있으면 시각적으로 어떻게 보이는지
- **약점/파괴 패턴**: 전투 안무에 반영할 정보

### 2단계: 디자인 생성

**출력 구조:**
```
[적 디자인 — 이름/코드명]

■ 위협 등급: 잡졸 / 중형 / 엘리트 / 보스
■ 질감 단계: Phase X

적 설명 (한글):
(크기, 외형, 재질, 무기/공격 방식, 위협적인 특징을 상세 서술)

Midjourney Prompt (영문):
"(1,000자 이내, --v 7)"

📏 글자수: 약 X자

고정 외형 키워드:
(이후 영상 컷에서 이 적을 묘사할 때 재사용할 키워드)

전투 메모:
- 공격 패턴: (주로 어떻게 공격하는지)
- 약점: (어디를 치면 무너지는지)
- 파괴 연출: (죽을 때 어떻게 보이는지 — 폭발/붕괴/소멸 등)
```

## Phase별 적 스케일

적의 디자인 복잡도와 위협감은 현재 Phase에 맞춘다.

- **Phase 1**: 단순한 형태, 둥글고 귀여운 편, 위협보다 장난스러운 느낌
- **Phase 2**: 뚜렷한 디자인, 날카로운 실루엣, 본격적인 적의 형태
- **Phase 3**: 디테일 풍부, 질감/재질 강조, 무게감과 존재감, 그림자와 조명으로 위압감
- **Phase 4**: 압도적 스케일, 초현실적 디자인, 기존 적들과 완전히 다른 차원의 존재감

## 위협 등급별 가이드

**잡졸 (Fodder)**
- 한 컷에 3~6체 동시 등장. 개별 디테일보다 실루엣 구별이 중요.
- 1~2초 만에 처리되므로 파괴 연출이 시원해야 한다 (파편, 스파크, 폭발).

**중형 (Medium)**
- 1:1 클래시가 가능한 수준. 무기나 공격 수단이 뚜렷해야 한다.
- 주인공과 칼날/주먹이 부딪히는 "충돌 포인트"를 디자인에 반영.

**엘리트 (Elite)**
- 잡졸과 확실히 다른 비주얼. 색상, 크기, 장식 등으로 차별화.
- 특수 능력이 시각적으로 드러나야 한다 (발광, 오라, 변형 등).

**보스 (Boss)**
- 화면을 압도하는 스케일. 캐릭터와 크기 대비가 극적.
- 약점 부위가 시각적으로 힌트되면 좋다 (발광 코어, 균열 등).
- 파괴 시 단계적 붕괴 연출 — 한 번에 죽지 않는 느낌.

## 주의사항

- 적 디자인은 단독 이미지로. 주인공과 함께 넣지 않는다 (합성 편의).
- **배경은 흰색/빈 배경** — `blank background` 또는 `white background`. 짧게.
- **텍스트 금지** — `no text` 하나면 충분. 과잉 금지 키워드(`no labels, no watermark, no annotations` 등)는 오히려 노이즈.
- **복장/장비/외형은 괄호 묶기로 나열.** 문장형 서술 금지. 미드저니는 짧은 키워드가 잘 먹힌다.
- 보스급은 전신이 보이는 풀샷 + 부위 클로즈업을 별도로 만들면 좋다.
