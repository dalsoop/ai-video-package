# 스토리보드 기획 (Storyboard Planner)

전체 영상의 흐름을 설계하고, 컷 배분과 Phase 전환 타이밍을 기획한다.

---

## 언제 사용하는가

- 프로젝트 시작 시 전체 구조를 잡을 때
- "전체 정리해줘" 요청 시
- 5컷마다 자동 리캡 시
- 새로운 챕터/시퀀스로 넘어갈 때

## 프로젝트 개요 템플릿

초기 인터뷰 완료 후 아래 형식으로 정리해서 사용자에게 보여준다.

```
📋 프로젝트 개요

■ 유형: (극장판/단편/숏폼/MV/트레일러)
■ 세계관: (장르 + 핵심 설정)
■ 예상 길이: (총 시간 / 총 컷 수)
■ 캐릭터: (등록된 캐릭터 목록 + 고정 키워드 요약)
■ 시작 Phase: Phase X

🎬 전체 흐름 (예상)
├ 도입부 (컷 #1~#N): Phase X — (핵심 내용)
├ 전개부 (컷 #N~#N): Phase X — (핵심 내용)
├ 클라이맥스 (컷 #N~#N): Phase X — (핵심 내용)
└ 마무리 (컷 #N~#N): Phase X — (핵심 내용)
```

## 컷 배분 가이드

프로젝트 유형별 권장 구조:

**숏폼/릴스 (15초~1분)**
- 1~4컷. 도입 없이 바로 액션. Phase 변화 없이 단일 Phase.
- 첫 1초에 시선을 잡아야 한다 — 컷 #1부터 액션 100%.

**트레일러/티저 (30초~2분)**
- 4~8컷. 하이라이트 장면만 모은 구조.
- Phase를 빠르게 오가며 스케일 변화를 보여줄 수 있다.
- 마지막 컷에서 최대 임팩트.

**단편 애니메이션 (3~10분)**
- 12~40컷. 3막 구조 가능 (도입-전개-클라이맥스).
- Phase 1→2→3 순차 성장. 마지막에 Phase 4 가능.
- 5컷마다 리캡.

**극장판 (30분 이상)**
- 120컷+. 여러 챕터로 분할.
- Phase 전환을 천천히. 각 Phase에 충분한 컷 수 배분.
- 10컷마다 리캡 권장.

## 5컷 리캡 템플릿

```
📍 흐름 정리 (컷 #1 ~ #5)

#1 [제목] — (한 줄 요약) | Phase X
#2 [제목] — (한 줄 요약) | Phase X
#3 [제목] — (한 줄 요약) | Phase X
#4 [제목] — (한 줄 요약) | Phase X
#5 [제목] — (한 줄 요약) | Phase X

현재 상태: (캐릭터 상태, 적 상황, 스토리 진행도)
다음 방향: (예상되는 전개)
```

## 전체 정리 템플릿

```
📋 전체 컷 흐름

완성된 컷:
#1 [제목] — (요약) | Phase X
#2 [제목] — (요약) | Phase X
...

남은 계획:
#N [제목] — (예상 내용) | Phase X
#N+1 [제목] — (예상 내용) | Phase X
...

등록된 에셋:
- 캐릭터: (목록)
- 배경: (목록)
- 적: (목록)
```

## Phase 전환 타이밍 판단

Phase 전환은 사용자가 직접 지시하지만, 아래 상황에서는 전환을 제안한다:
- 새로운 적 등급이 등장할 때 (잡졸→보스)
- 캐릭터가 새 무기/능력을 얻었을 때
- 환경이 극적으로 바뀔 때 (도시→이계, 지상→공중)
- 사용자가 "분위기 바꿔" "더 진지하게" 등 톤 변화를 요청할 때
