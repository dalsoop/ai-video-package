# 스토리보드 기획 (Storyboard Planner)

전체 영상의 흐름을 설계하고, 컷 배분과 Phase 전환 타이밍을 기획한다.

---

## 핵심 개념: 스토리 먼저, 컷 분할 나중

1. **먼저 전체 스토리를 길게 짠다.** 사용자의 러프한 아이디어를 받아서 종교하고 긴 시나리오를 작성한다.
2. **그 다음 15초 단위로 분할한다.** 1컷 = Seedance 15초 영상 1개.
3. 컷 간 연결이 어색하지 않게 배분한다.

사용자가 러프하게 "이런 느낌으로 해줘"라고 말해도 알아서 체계적인 시나리오로 발전시킨다.

---

## 언제 사용하는가

- 프로젝트 시작 시 전체 구조를 잡을 때
- "전체 정리해줘" 요청 시
- 5컷마다 자동 리캡 시
- 새로운 챕터/시퀀스로 넘어갈 때

## 프로젝트 개요 템플릿

초기 인터뷰 완료 후 아래 형식으로 정리해서 사용자에게 보여준다.

```
📋 프로젝트 개요

■ 유형: (극장판/단편/숏폼/MV/트레일러)
■ 세계관: (장르 + 핵심 설정)
■ 예상 길이: (총 시간 / 총 컷 수)
■ 캐릭터: (등록된 캐릭터 목록 + 고정 키워드 요약)
■ 시작 Phase: Phase X

🎬 전체 흐름 (예상)
├ 도입부 (컷 #1~#N): Phase X — (핵심 내용)
├ 전개부 (컷 #N~#N): Phase X — (핵심 내용)
├ 클라이맥스 (컷 #N~#N): Phase X — (핵심 내용)
└ 마무리 (컷 #N~#N): Phase X — (핵심 내용)
```

## 컷 배분 가이드

**1컷 = 핵심 동작 1개 + 카메라 무빙 1개.** Seedance 최대 15초.

프로젝트 유형별 권장 구조:

**숏폼/릴스 (15초~1분)**
- 2~8컷. 도입 없이 바로 액션. Phase 변화 없이 단일 Phase.
- 첫 1초에 시선을 잡아야 한다 — 컷 #1부터 동작 시작.

**트레일러/티저 (30초~2분)**
- 4~16컷. 하이라이트 장면만 모은 구조.
- Phase를 빠르게 오가며 스케일 변화를 보여줄 수 있다.
- 마지막 컷에서 최대 임팩트.

**단편 애니메이션 (3~10분)**
- 24~80컷. 3막 구조 가능 (도입-전개-클라이맥스).
- Phase 1→2→3 순차 성장. 마지막에 Phase 4 가능.
- 5컷마다 리캡.
- **기승전결은 컷 배분으로.** 하나의 컷에 기승전결을 넣지 않는다.
- 예: 5분 영상 = 40컷. 도입 6컷, 전개 16컷, 클라이맥스 12컷, 마무리 6컷.

**극장판 (30분 이상)**
- 120컷+. 여러 챕터로 분할.
- Phase 전환을 천천히. 각 Phase에 충분한 컷 수 배분.
- 10컷마다 리캡 권장.

## 스토리 작성 원칙

- 사용자가 러프하게 말해도 길고 구체적인 시나리오로 발전시킨다.
- 액션 영상의 핵심: **지루할 틈을 주지 않는다.**
- 다채로운 장면이 계속 나오면 액션만 계속 나와도 덜 지루하다.
- 질감/분위기 변화를 컷마다 조금씩 주면 시각적 지루함 방지.
- **전형적인 애니메이션 구조 금지**: 이동→도착→대치→전투 같은 느린 구조 대신, 바로 전투→즉시 다음 전투.
- 적 처치 → 다음 적 즉시 등장. 사이에 이동/감성 씬 없음.
- 보스급 적을 상대할 때도 인물이 오래 누워있거나 주저앉는 장면 최소화. 바로 일어나서 반격.

## 컷 간 전환 원칙

- 한 전투가 끝나면 **바로** 다음 전투. 이동 씬 금지.
- 환경 전환은 액션 도중에 자연스럽게. 예: 카메라가 360도 돌면 배경이 바뀌어 있다.
- 캐릭터 레벨업/장비 변화는 3초 이내. 목검→진검, 맨손→마법 등.
- 시간 경과/성장은 질감 변화 + 캐릭터 외형 변화로 표현. 나레이션/자막 불필요.

## 5컷 리캡 템플릿

```
📍 흐름 정리 (컷 #1 ~ #5)

#1 [제목] — (한 줄 요약) | Phase X
#2 [제목] — (한 줄 요약) | Phase X
#3 [제목] — (한 줄 요약) | Phase X
#4 [제목] — (한 줄 요약) | Phase X
#5 [제목] — (한 줄 요약) | Phase X

현재 상태: (캐릭터 상태, 적 상황, 스토리 진행도)
다음 방향: (예상되는 전개)
퀄리티 체크: (미드저니 키컷 재생성 필요 여부)
```

## 전체 정리 템플릿

```
📋 전체 컷 흐름

완성된 컷:
#1 [제목] — (요약) | Phase X
#2 [제목] — (요약) | Phase X
...

남은 계획:
#N [제목] — (예상 내용) | Phase X
#N+1 [제목] — (예상 내용) | Phase X
...

등록된 에셋:
- 캐릭터: (목록)
- 배경: (목록)
- 적: (목록)

퀄리티 상태:
- 마지막 미드저니 키컷: 컷 #N
- 연속 Seedance 연결 횟수: X회
- 키컷 재생성 권고: (예/아니오)
```

## Phase 전환 타이밍 판단

Phase 전환은 사용자가 직접 지시하지만, 아래 상황에서는 전환을 제안한다:
- 새로운 적 등급이 등장할 때 (잡졸→보스)
- 캐릭터가 새 무기/능력을 얻었을 때
- 환경이 극적으로 바뀔 때 (도시→이계, 지상→공중)
- 사용자가 "분위기 바꿔" "더 진지하게" 등 톤 변화를 요청할 때
- **Seedance 연속 사용으로 퀄리티가 떨어졌을 때** — Phase 전환 + 미드저니 키컷 재생성의 좋은 타이밍

**Phase 전환 시에는 반드시 미드저니 키컷을 새로 만든다.** 질감이 확 바뀌는 순간이므로 Seedance 마지막 프레임을 이어쓰면 안 된다.
